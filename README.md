#BrainRS
Simple BrainF\*ck interpreter written in Rust.

#How to build
```bash
$ git clone --single-branch rs https://github.com/d-dorazio/mindblown brainrs
$ cd brainrs/
$ cargo build --release
```

#How to use
Command line interpreter:
```bash
$ cargo run
```

File interpreter:
```bash
$ cargo run bf/helloworld.bf
```
